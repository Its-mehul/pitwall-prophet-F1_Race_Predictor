# Assets Folder

## Required Images

### F1 Logo
- **File**: `f1_logo.png`
- **Location**: Place directly in `assets/` folder
- **Format**: PNG with transparency preferred
- **Recommended size**: 1000x1000px or larger
- **Usage**: Translucent background on landing page

### Team Logos
- **Location**: `assets/teams/` folder
- **Naming convention**: `{team_name_lowercase_with_underscores}.png`
- **Format**: PNG with transparency preferred
- **Recommended size**: 200x200px (square)
- **Usage**: Displayed on starting grid cards

## Fallback Behavior
If a team logo is not found, the image will simply not display (hidden), and the driver card will still show with driver name and team name.
